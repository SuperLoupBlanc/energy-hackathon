# Energy Hackathon

Prototype de site web statique pour un hackathon sur le thème de l'énergie.

## Objectif

Aider les utilisateurs à visualiser leur consommation énergétique, comprendre les pics et recevoir des recommandations simples.

## Fonctionnalités

- Page de présentation claire.
- Démo interactive avec estimation de consommation.
- Dashboard animé en JavaScript.
- Design responsive pour mobile et desktop.
- Compatible GitHub Pages.

## Structure

```text
energy-hackathon/
├─ index.html
├─ style.css
├─ script.js
├─ assets/
└─ README.md
```

## Lancer en local

Ouvre simplement `index.html` dans ton navigateur.

Option avec Python :

```bash
python -m http.server 8000
```

Puis va sur :

```text
http://localhost:8000
```

## Déployer sur GitHub Pages

1. Crée un dépôt GitHub nommé `energy-hackathon`.
2. Envoie ces fichiers dans le dépôt.
3. Va dans `Settings > Pages`.
4. Sélectionne `Deploy from a branch`.
5. Choisis `main` et `/root`.
6. Récupère l'URL publique GitHub Pages.

URL attendue :

```text
https://tonpseudo.github.io/energy-hackathon/
```

## À personnaliser

- Le nom du projet.
- Les membres de l'équipe.
- Le lien GitHub dans `index.html`.
- Les textes sur le problème et la solution.
- Les chiffres d'impact si tu as de vraies données.
